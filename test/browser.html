🚀 Console-v2 增强版日志工具

一个功能强大的前端开发控制台工具，支持彩色、图片、多格式日志输出

🎨 彩色日志功能

支持丰富的颜色和样式，让日志更加美观易读

🖼️ 图片日志功能

在控制台中显示图片，支持单张、批量、网格等多种模式

🎭 SVG 功能

支持SVG图形、图标、进度条、图表等丰富的可视化元素

📊 格式化工具

提供表格、进度条、时间戳、分组等格式化功能

🎯 组合使用示例

展示如何组合使用各种功能创建复杂的日志输出

🎨 颜色系统

  • 16种预定义颜色
  • 自定义颜色支持
  • 背景色和渐变
  • 文字样式组合

🖼️ 图片功能

  • 单张图片显示
  • 批量图片加载
  • 图片网格布局
  • 自适应尺寸

🎭 SVG 支持

  • 自定义SVG图形
  • 预定义图标库
  • 进度条组件
  • 图表生成器

📊 格式化工具

  • 数据表格显示
  • 进度条组件
  • 时间戳格式化
  • 分组日志管理

💻 使用方法

// 导入工具 import console from './lib/index.js'; // 彩色日志 console.log('%c红色文字', console.color(console.colors.red)); console.success('成功消息'); // 图片日志 await console.img('https://example.com/image.jpg', { width: 200, height: 100 }); // SVG功能 console.svgIcon('star', 32, console.colors.orange); console.svgProgressBar(75); // 格式化工具 console.table(data, { title: '数据表' }); console.progress(65, 100);
